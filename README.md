# .NET Restaurant Management System

## 🚀 Author
Bojan Brankovic 

## Project Overview
This project is a Restaurant Management System developed using .NET technologies. It is designed to simulate real-world restaurant operations, including order management, menu handling, and basic administrative workflows.

The goal of this project is to demonstrate backend logic, system organization, and practical business use cases within a restaurant environment.

## Features
- Menu management (add/edit/remove items)
- Order creation and processing
- Table management
- Basic user/admin functionalities
- Order tracking and status updates
